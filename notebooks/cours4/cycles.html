4.5 Cycles

Write a node node cycle(x: int) returns (y: int) where y starts at the absolute value of x and cycles through n,n-1,\ldots,0 where n is the initial value of y.

Now write a contract for cycle, which checks the following properties:

  1. y is always non-negative
  2. y is bounded above by its initial value
  3. Unless the current value of y equals the initial value of y, it is smaller than its previous value
  4. If y was initially 0, then y has always been 0
  5. The output of cycle depends only on the initial value of x
