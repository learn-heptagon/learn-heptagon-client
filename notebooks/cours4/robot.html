4.6 Voice-controlled Robot

Write the nodenode robot_control(move_cmd, stop_cmd, obst: bool; dir, dist: int) returns (rotate, go: bool)

The robot moves using two actuators:

  • rotate which turns the robot clock-wise
  • go which moves the robot forward
It has the following inputs:
  • move_cmd means the robot heard the command "Move"
  • stop_cmd means the robot heard the command "Stop"
  • obst is true if there is an obstable between the robot and the speaker
  • dir is the angle (in degrees, between 0 and 359) between the heading of the robot and the speaker (0 means straight ahead, 90 means to the right, etc)
  • dist is the distance (in cm) between the robot and the speaker

Then, write a contract for the following requirements:

  1. The robot never advances and rotates at the same time.
  2. If an obstacle is detected, the robot does not move.
  3. The robot does not move if it is or comes within half a meter of the speaker.
  4. If a speaker gives a “Stop” command, the robot stops and does not move until the “Move” command is given again.
  5. If the robot receives only a “Move” command by a speaker more than 1/2 meter away and there is no obstacle in the way, then the robot will start moving.
  6. In the absence of obstacles, a moving robot will keep moving until it is within 1/2 meter of the speaker or it receives a “Stop” command.
