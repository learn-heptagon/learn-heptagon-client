4.2 Basic Operators

Before we start specifying nodes, we will write a few logical operators that we may reuse later (you will have to copy and paste these definitions in other editors). Implement the following nodes:

  • first_int and first_bool which constantly return the initial value of their input stream
  • always which returns true if its input has always been true (seen in course 1)
  • exist which returns true if its input has ever been true
