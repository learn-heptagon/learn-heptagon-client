1.5 Stopwatch

Write a node stopwatch (on_off, rs, freeze: bool) returns (time: int) where:

  • on_off starts and stops the stopwatch
  • rs resets the stopwatch (if not running)
  • freeze freezes the display (if running)
Hint: you should reuse previous nodes and local variables.
  • running: bool, a swtch instance
  • frozen: bool, a swtch instance
  • cpt: bool, a counter instance
