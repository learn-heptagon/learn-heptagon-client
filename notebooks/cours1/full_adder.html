1.2 Programming 1-bit adders

In this exercise, we will use Lustre to implement a 1-bit adder. As a first approximation, a 1-bit-adder takes two boolean values a and b, and returns their sum s and carry co. However, to facilitate chaining 1-bit adders (to construct multiple-bits adders), we add a third operand c to the sum. The truth table for this operator is presented below.
