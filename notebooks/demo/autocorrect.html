Autocorrect

Now that you tried your node, you can check if you actually got it right ! Click on the "Autocorrect" tab. If your node confirms to the spec, you should see a green text. If not, you should see a red text with a counterexample. The autocorrect process is not syntactic, so feel free to rewrite your code, as long as it stays equivalent e.g. using -> and pre instead of fby, separate the definition of x using an intermediate variable and equation, etc.
More on how it works in the rest of the presentation.
