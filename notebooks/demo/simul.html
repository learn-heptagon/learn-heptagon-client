Once you have completed the node above, two tabs titled "Simulator" and "Autocorrect" should appear. Let's start with the simulator.

Simulator

For this node, the simulator's interface is just a normal chronogram which relates the input (line x) to the output (s). Before clicking "Step", you need to provide an input value for x in the opened input field. You may also specify a Lustre expression to compute x in the editor right to "x = ". This expression may depend on x only, so it could be e.g. 0 fby (x + 1).
